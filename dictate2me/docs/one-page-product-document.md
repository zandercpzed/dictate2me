# Dictate2Me — One-Page Product Document

## Visão do Produto

Plugin Obsidian para transcrição de voz 100% local, privacidade total, integração nativa e correção inteligente.

## MVP

- Captura de áudio local
- Transcrição offline
- Correção opcional
- API para integração

## Público-alvo

Usuários Obsidian que desejam transcrever notas por voz sem depender de serviços externos.

## Diferenciais

- 100% offline
- Correção via LLM local
- Integração nativa Obsidian

## Roadmap

- Fase 0: Setup
- Fase 1: Captura e transcrição
- Fase 2: Correção
- Fase 3: Integração avançada
